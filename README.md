# Arquitectura Web para Consultas Concurrentes a LLMs

## Descripción
Proyecto de Final de Grado (UPC-EETAC) enfocado en la reducción de la fragmentación de modelos de lenguaje (ChatGPT, DeepSeek, Llama) mediante una arquitectura asíncrona.

## Objetivos
- Comparativa en tiempo real de latencia y calidad.
- Análisis de eficiencia energética y sostenibilidad.
- Implementación de concurrencia mediante FastAPI.

## Stack Tecnológico
- **Backend:** Python + FastAPI
- **Frontend:** Streamlit
- **Modelos:** OpenAI API, DeepSeek API, Ollama (Llama 3)
- **Infraestructura:** Git + GitHub (Futuro: AWS)
